2. 자주 쓰는 명령어

* git init
   - git은 시작부터 다 트랙킹하고 있진 않음.

* git add 파일명
  - git은 이제부터 파일 트랙킹(버전관리)하고 있음.
  - unStaged git add 전 상태로 돌리고 싶다면..
  - git rm --cached <file>, git rm --staged <file>

* git commit 
  - git이 tracking하고 있는 파일 변화 tracking한거 다 모으고 있음.
  - 지금까지 모은거(git add .)로 버전 만들어줘
